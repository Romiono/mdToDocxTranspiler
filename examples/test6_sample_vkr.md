# СОДЕРЖАНИЕ

# ВВЕДЕНИЕ

Актуальность темы исследования обусловлена необходимостью комплексного анализа инвестиционного потенциала регионов Российской Федерации в условиях современных экономических вызовов. Краснодарский край занимает одно из ведущих мест среди субъектов Федерации по объёму привлечённых инвестиций [1, с. 5].

Целью данной работы является разработка методических подходов к оценке инвестиционного потенциала региона на примере Краснодарского края.

Для достижения поставленной цели необходимо решить следующие задачи:

- изучить теоретические основы инвестиционного потенциала региона;
- провести анализ текущего состояния инвестиционной деятельности;
- разработать рекомендации по повышению инвестиционной привлекательности.

Объектом исследования является Краснодарский край как субъект Российской Федерации. Предметом исследования выступают экономические отношения, возникающие в процессе формирования инвестиционного потенциала региона.

# 1 Теоретические основы инвестиционного потенциала региона

## 1.1 Сущность и содержание понятия «инвестиционный потенциал»

Понятие «инвестиционный потенциал» является одним из ключевых в современной региональной экономике. Под инвестиционным потенциалом региона понимается совокупность объективных условий, предпосылок и ресурсов, определяющих возможности региона по привлечению и эффективному использованию инвестиций [2].

Инвестиционный потенциал включает следующие составляющие:

- природно-ресурсный потенциал;
- трудовой потенциал;
- производственный потенциал;
- инфраструктурный потенциал;
- инновационный потенциал;
- финансовый потенциал.

Для количественной оценки инвестиционного потенциала используется следующая формула:

$$IP = \sum_{i=1}^{n} w_i \cdot P_i$$ (1.1)

где $IP$ – интегральный показатель инвестиционного потенциала;
- $w_i$ – весовой коэффициент i-го компонента;
- $P_i$ – нормализованное значение i-го компонента потенциала;
- $n$ – количество компонентов потенциала.

## 1.2 Классификация факторов инвестиционного потенциала

Факторы, определяющие инвестиционный потенциал региона, принято делить на две группы:

а) объективные факторы:
1) географическое положение региона;
2) природно-климатические условия;
3) обеспеченность природными ресурсами.

б) субъективные факторы:
1) качество государственного управления;
2) уровень развития инфраструктуры;
3) инвестиционный климат.

# 2 Анализ инвестиционного потенциала Краснодарского края

## 2.1 Общая характеристика инвестиционной деятельности региона

Краснодарский край является одним из наиболее инвестиционно привлекательных регионов России. По данным Росстата, объём инвестиций в основной капитал в 2023 году составил более 600 млрд рублей [3, с. 47].

# 3 Рекомендации по повышению инвестиционной привлекательности региона

## 3.1 Основные направления совершенствования инвестиционной политики

На основе проведённого анализа можно сформулировать следующие рекомендации по повышению инвестиционной привлекательности Краснодарского края:

- совершенствование нормативно-правовой базы в сфере инвестиционной деятельности;
- развитие инфраструктуры поддержки инвесторов;
- создание специализированных инвестиционных площадок;
- формирование системы мониторинга инвестиционного климата.

Ожидаемый экономический эффект от реализации предложенных мероприятий рассчитывается по формуле:

$$E = \sum_{t=1}^{T} \frac{CF_t}{(1 + r)^t} - I_0$$ (3.1)

где $E$ – чистый приведённый эффект, млрд руб.;
- $CF_t$ – денежный поток в периоде t, млрд руб.;
- $r$ – ставка дисконтирования;
- $T$ – горизонт расчёта, лет;
- $I_0$ – первоначальные инвестиции, млрд руб.
